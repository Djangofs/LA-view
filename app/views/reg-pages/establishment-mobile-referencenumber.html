Mobile establishment reference number

What is an establishment?

An establishment is the location of your food business. If it is a mobile food business, please use the location where it is normally stored overnight.

What is a mobile establishment reference number?

A mobile establishment reference number is a number that is associated with your moving premise. For example vehicle registration number if it is a food truck or street trading license number if it is a market stall.
